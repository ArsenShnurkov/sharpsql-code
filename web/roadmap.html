Milestone 1

In the first milestone the creation of a connection with the ADO.NET driver for the MS SQL Server should possible.

Milestone 2

The base SQL feature like CREATE TABLE, DROP TABLE, SELECT, INSERT, UPDATE and DELETE should be possible.

Milestone 3

The security system and permissions will be implemented. This means you will need a user account to create a connection.
